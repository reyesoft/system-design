Flex layout

En el proyecto hay clases que nos permiten utilizar flex para poder alinear elementos hijos dentro de un elemento padre

El recurso se encuentra en la ruta "src/assets/styles/_object.scss".

Las clases son las siguientes:

Alinear los elementos en fila

flex-row-center-center: los elementos se ubican en fila y se alinean completamente al centro

flex-row-end-center: horizontal: derecha, vertical: centrado

flex-row-spacebetween-center: horizontal: ocupan todo el espacio, vertical: centrado

flex-row-start-center: horizontal: izquierda, vertical: centrado

flex-row-center-start: horizontal: centrado, vertical: arriba

flex-row-end-center: horizontal: derecha, vertical: centrado

Alinear los elementos en columnas

flex-column-center-center: horizontal: centrado, vertical: centrado

flex-column-start-center: horizontal: centrado, vertical: arriba

flex-column-center-end: horizontal: derecha, vertical: centrado

flex-column-start-start: horizontal: izquierda, vertical: arriba

flex-column-spacebetween-center: horizontal: centrado, vertical: ocupa todo el espacio
