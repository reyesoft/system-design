Grid responsive

Dentro del proyecto nos encontramos con una serie de clases, las cuales tiene una función similar a la clase rs-wrapper, la diferencia es que tenemos una clase rs-wrapper-responsive, la cual nos permite, por ejemplo tener un elemento padre, al cual le definimos un número de columnas fijo, y medida que hacemos resize del viewport las columnas solas se acomodan de acuerdo al espacion disponible, sin la necesidad de utilizar media queries.

El recurso lo podemos encontrar en "src/assets/styles/_object.scss"

Agregar la clase rs-wrapper-responsive al elemento padre, pero en este caso tambien le definiremos un numero de columnas, para poder observar como funciona la clase

  
    <div class="father-element rs-wrapper-responsive col5-responsive">
      <div class="child-element">Child One</div>
      <div class="child-element">Child two</div>
      <div class="child-element">Child Three</div>
      <div class="child-element">Child Four</div>
      <div class="child-element">Child Five</div>
      <div class="child-element">Child Six</div>
    </div>

En este caso observamos que esta dividido en 4 columnas, ya que el calculo para las columnas esta hecho sobre un width 1200px
